### Таблица адресов 
Адрес план составлен по следующей схеме:
10.Dn.Sn.X/31

Dn- Номер адресного простаранства в ЦОД №1 , (0-7)
0-1 сеть Loopback интерфейсо (Lo1,Lo2)
Sn- номер Spine,
X- значение по порядку.
